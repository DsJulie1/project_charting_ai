STT 기반 상담 내용 차팅 자동화

이 프로젝트의 핵심은 한의사와 환자의 대화 내용을 담은 음성 파일을 텍스트로 변환하고 다음 5가지 주요 정보를 찾아 차트 작성을 자동화하는 것 입니다.

  • 1. 주소증 : 환자가 어떤 부분이 불편해서 한의원에 방문했는지 기입
  • 2. 발병일 : 주소증이 언제 시작되었는지 기입
  • 3. 과거력 : 환자가 과거에 앓았던 질환, 과거 수술 및 치료 이력, 과거 동일한 증상이 있었는지에 대해 기입(암, 고혈압, 당뇨, 고지혈증 등 포함)
  • 4. 가족력 : 환자의 증상과 비슷한 증상을 보이는 가족이 있었다면 기입
  • 5. 현병력 : 현재 한의원에 방문하기 전까지 어떤 과정이 있었는지 기입

Why you should do this project

효율성 향상: 복잡한 차트 작성 과정을 자동화함으로써, 한의사들이 보다 많은 시간을 직접적인 치료에 집중할 수 있게 됩니다.

정확성 증가: 음성 인식 기술을 활용하여 직접적으로 환자의 말을 차트에 기록함으로써, 정보의 정확성이 향상될 것입니다.

서비스 향상: 환자의 상태를 더 정확하고 빠르게 파악할 수 있게 되어, 환자에게 더욱 개선된 진료 서비스를 제공하는데 도움이 될 것입니다.

데이터 분석 가능: 음성에서 추출된 텍스트 데이터를 기반으로, 환자의 상태 변화, 질병 패턴 등을 분석하는데 사용할 수 있습니다. 이는 장기적으로는 치료 방안 개발이나 예방 전략 수립에 도움을 줄 수 있습니다.

FAQ

Whisper?

Whisper는 OpenAI에서 개발한 자동 음성 인식(Automatic Speech Recognition, ASR) 시스템입니다. 이 시스템은 음성 데이터를 텍스트 데이터로 변환하는 역할을 합니다. Whisper는 다양한 언어와 방언, 그리고 다양한 환경 속에서의 음성을 인식할 수 있도록 설계되었으며, 이를 통해 음성 기반의 애플리케이션에 사용될 수 있습니다.

ChatGPT?

ChatGPT는 대화형 인공지능 모델로, 사용자의 입력에 대해 자연스럽고 의미있는 응답을 생성하도록 설계되었습니다. 이 모델은 특정 문맥에서 어떤 단어가 적절한지 예측하거나, 사용자의 질문이나 요청에 응답하는 등의 작업을 수행합니다. ChatGPT는 OpenAI의 GPT-3 모델을 기반으로 하며, 다양한 대화 시나리오에 적용할 수 있습니다.
